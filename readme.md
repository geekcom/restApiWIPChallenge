# WIP API

---
## Products

`POST`
```sh
http://localhost:8000/api/v1/products
```

`GET`
```sh
http://localhost:8000/api/v1/products/{product_id}
```

`PUT`
```sh
http://localhost:8000/api/v1/products/{product_id}
```

`DELETE`
```sh
http://localhost:8000/api/v1/products/{product_id}
```
